Challenges
  • Generali S.p.a. - Data Challenge 2020 [Code: WRITE ME A MAIL IF YOU ARE INTERESTED, Report: ]

    Supervised Churn Classification challenge on a insurance policy dataset provided by Generali Italia S.p.a. I proposed and implemented an original architecture, combining in a hierarchical structure several binary classifiers through ensemble learning. My model won the first prize in the qualitative ranking (best report and code) and it produced the 7th (2th) best prediction on the private (public) test set out of 280 participants.

  • EPFL - Higgs Boson Challenge 2020 [Code: , Report: ]

    In a team of 3 we solved the Higgs Boson Machine Learning Challenge: a classification problem on a huge dataset simulated by the ATLAS experiment from CERN. Getting an accuracy on the prediction of 0.841 and a F1 Score of 0.757 we secured the 7-th place out of 277 teams in the final ranking on AIcrowd (not being allowed to use external libraries, we have implemented ALL the algorithms starting from numpy).

  • Oracle Labs and Politecnico di Milano - Graph Machine Learning Contest 2019 [Code: , Report: ]

    In a team of two, I won an Oracle Labs-backed contest on Kaggle. Our task was to perform multi-label vertex classification on a protein-protein interaction network generating a powerful embedding.

  • Politecnico di Milano - Machine Learning for Networking Contest 2019 [Code: ]

    I won the 1st Place in the Kaggle final challenge of Machine Learning for Networking course provided by Polytechnic of Milan, using supervised learning to predict the satisfaction of the customers of a mobile cellular network operator.
